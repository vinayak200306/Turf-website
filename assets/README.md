# FIELD DOOR Assets

Place the production hero media in this folder with these filenames:

- `hero-turf.mp4` for the autoplay hero background video
- `hero-poster.svg` or a replacement poster image for the video fallback

Current implementation notes:

- The page expects `./assets/hero-turf.mp4`
- The page uses `./assets/hero-poster.svg` as the current poster fallback
- Featured arena cards already render with CSS gradient fallbacks, so extra venue images are optional
